# Downloader music spotify V1.2

# Installation & requirements

Have node js downloaded on your computer.

```bash
git clone https://github.com/LittleKidd0/downloader-music-spotify
```
```bash
cd downloader-music
```
```
chmod +x install.sh && ./install.sh
```
Windows
```bat
install.bat
```
